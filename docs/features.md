# 功能说明

## 图像查看

支持 OpenCV `cv::Mat` / `cv::Mat_<T>` 以及多种原生内存图像格式，适合图像算法开发、视觉检测和调试场景。

## Mask Overlay

把单通道图像作为透明叠加层显示在当前图像上，便于检查阈值分割、缺陷区域和 ROI 结果。

## Contour / Geometry

支持 OpenCV 点、矩形、旋转矩形、单轮廓和多轮廓，适合检查 `findContours`、几何拟合和定位结果。

## Plot / Histogram

支持一维数值序列、折线图、散点图和直方图，用于调试投影、分布、灰度统计和中间数值结果。

## Export

可导出完整显示结果，方便保存调试证据和分享问题复现。
